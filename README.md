# TailwindCSS Coffee Style by AdStar

## Requirements

* **nodejs >= v14.7.0**
* **npm >= v6.14.7**
* **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn build:css
$ yarn start:dev
```
